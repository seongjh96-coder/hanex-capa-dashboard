사용 매뉴얼

센터 담당자가 작성하고, 운영 관리자가 전체 CAPA와 센터별 상세 현황을 분석하는 흐름입니다.

1. 센터 선택

대시보드 상단의 센터 슬라이서에서 센터를 선택하면 상세 CAPA와 고객사 점유 현황이 바뀝니다.

2. CAPA 입력

센터별 입력 메뉴에서 공간 분류별 가능 CAPA, 실사용 CAPA, 화주사 점유 CAPA를 입력합니다.

3. 도면 업로드

대시보드의 센터 도면 점유도에서 센터별 도면 이미지를 업로드합니다.

4. 도면 점유 설정

영역 추가 버튼으로 고객사 구역을 만든 뒤 도면 셀을 클릭하거나 드래그해 ㄴ자, ㅁ자 형태까지 표시합니다.

5. 공실 방어 확인

여유 CAPA 카드와 노란 강조 영역을 확인해 센터별 추가 영업 가능 물량을 판단합니다.

6. 자료 공유

CSV 내보내기로 현재 CAPA 입력 데이터를 내려받아 회의자료나 취합용 파일로 활용합니다.
